# Андрей Соколов — Актёрское портфолио

Интерактивное портфолио актёра, созданное с помощью React, Vite и Tailwind CSS.

## Демонстрация

Посмотреть демо можно по ссылке: https://rodion-korolev.github.io/actor-portfolio-interactive-design/

## Технологии

- **React 19** — современный UI фреймворк
- **TypeScript** — типобезопасный JavaScript
- **Vite** — быстрый сборщик проектов
- **Tailwind CSS** — утилитарный CSS фреймворк
- **Framer Motion** — анимации
- **Lucide React** — иконки

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка проекта
npm run build
```

## Развертывание

Проект автоматически развертывается на GitHub Pages при каждом пуше в ветку `main`.

Для ручного развертывания:

```bash
# Установка gh-pages (если еще не установлено)
npm install --save-dev gh-pages

# Развертывание на GitHub Pages
npm run deploy
```

## Структура проекта

```
src/

├── App.tsx        # Главный компонент
```
