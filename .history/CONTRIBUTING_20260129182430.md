# Contributing to Actor Portfolio

Благодарим за интерес к проекту! Вот как вы можете внести свой вклад:

## Как внести изменения

1. **Форкните репозиторий**
2. **Создайте ветку** для вашей фичи (`git checkout -b feature/amazing-feature`)
3. **Сделайте коммит** (`git commit -m 'Add amazing feature'`)
4. **Отправьте в свой форк** (`git push origin feature/amazing-feature`)
5. **Создайте Pull Request`

## Требования к коду

- Используйте TypeScript для типобезопасности
- Следуйте принципам чистого кода
- Компоненты должны быть переиспользуемыми
- Используйте Tailwind CSS для стилей
- Пишите осмысленные коммиты

## Стиль кода

- Используйте Prettier для форматирования
- Следуйте конвенциям React/TypeScript
- Компоненты называйте в PascalCase
- Файлы стилей — в camelCase

## Тестирование

Перед созданием PR убедитесь что:

- Сборка проходит успешно (`npm run build`)
- Все линтеры проходят
- Функциональность работает корректно

## Вопросы

Если у вас есть вопросы, создайте Issue в репозитории.
